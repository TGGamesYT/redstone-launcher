# Redstone Launcher
This is the source of the unofficial Minecraft launcher; Redstone Launcher.

Redstone Launcher is an Electron based app, meaning it runs on node.js

to run the app, clone the repo and run the app command

```git clone https://github.com/tggamesyt/redstone-launcher.git```

```npm run app```

To build the app, clone the repo and run:

```npm run build```
